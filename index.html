Programpedia

Un lenguaje de programación es un lenguaje formal que le proporciona a un programador, la capacidad de escribir (o programar) una serie de instrucciones o secuencias de órdenes en forma de algoritmos con el fin de controlar el comportamiento físico o lógico de un sistema informático, de manera que se puedan obtener diversas clases de datos o ejecutar determinadas tareas.

La programación es una disciplina que se basa en diferentes lenguajes. Cada uno de estos lenguajes tiene características únicas que lo diferencian del resto, y estas características los hacen más adecuados para una tarea u otra. Veamos qué tipos de lenguajes existen.

Tipos de lenguajes de programación

  • Lenguajes de medio nivel
  • Lenguajes de alto nivel
